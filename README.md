## 💻 技术栈与分部

社团由一群热衷于“把代码写在大地上”的大佬组成，目前划分为三个核心实验室：

### 1. 算法研究室 (Algorithm Lab)

* **主攻：** 数据结构、动态规划、图论、模型调优。
* **语言：** `C++` / `Python` / `Java` / `Rust`

### 2. 麒麟安全组 (Security Squad)

* **主攻：** Web 安全、Pwn、逆向工程、AI 安全。
* **战场：** 鹏云杯、各类 CTF 联赛。
